# darkdmtr

> Personal music and creative portfolio site for darkdmtr (Dmitry Markelov) — hip-hop producer, rapper, and AI engineer based in Carlsbad, CA.

**Live Site:** [darkdmtr.com](https://darkdmtr.com) | Custom domain via GitHub Pages | HTML/CSS/JS

---

## What It Is

This is the personal artist site for **darkdmtr** — a self-built HTML/CSS/JavaScript site hosted on GitHub Pages with a custom CNAME domain. It serves as a hub for:

- Music releases and track streaming
- Downloadable tracks
- Album art and visual brand assets
- Personal stats / analytics page

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| Deployment | GitHub Pages (custom CNAME domain) |
| Domain | darkdmtr.com (custom domain) |
| Assets | Music tracks, album artwork |
